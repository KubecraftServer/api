# Hello there 👋

## Getting Started 👨‍💻
### Starting Locally 🖥
1. Clone repository
```shell
$ git clone git@github.com:kubecraftserver/api.git
$ cd launcher
```
2. Edit `.env` file.
3. Install dependencies
#### We strongly recommend using `yarn` insted of `npm`
```shell
$ yarn install
```
4. Start in dev mode
#### We strongly recommend using `yarn` insted of `npm`
```shell
$ yarn run dev
``` 
5. Build
#### We strongly recommend using `yarn` insted of `npm`
```shell
$ yarn run build
```
